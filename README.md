# How to install for development:

## Requirements:
NodeJS: v12.16.3
Python: 3.6

## Front End:
Go to git repository in the terminal

Then go to the g34-covid-analyser folder

Then run 'npm install'

React (For UI)
victory (for charts)
Grommet (For UI Components)

## Back End:

python3 main.py
